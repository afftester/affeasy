## What is a Tag?

A tag is a label that you can assign to a link. Tags are color-coded and can be created and assigned to a link in just a few clicks.

With tags, you can:

1. Organize your links by campaigns, clients, or any other categories you can think of.
2. Filter your links by tags and get a shareable link to the filtered results.
3. **Filter your analytics by tags** to see how your campaigns are performing.

## How to create a Tag

To create a tag, follow these steps:

1. Navigate to the **Tags settings page** in your AffEasy workspace.

<Frame>
  <img
    className="rounded-lg border border-gray-100"
    src="/images/tags/tags-settings-page.png"
    alt="Tags settings page"
    width="600"
    height="400"
  />
</Frame>

2. Click on the "Create Tag" button to open the tag creation modal.
3. Enter a name for your tag and choose a color.

<Frame>
  <img
    className="rounded-lg border border-gray-100"
    src="/images/tags/tag-creation-modal.png"
    alt="Tag creation modal"
    width="400"
    height="300"
  />
</Frame>

You can also create new tags directly inside the **AffEasy link builder**:

1. After **creating a link**, click on the tag input field.
2. Type in the name of the tag you want to create.
3. If the tag doesn't exist, press `Enter ↩` to create a new tag.

<Frame>
  <img
    className="rounded-lg border border-gray-100"
    src="/images/tags/create-tag-in-builder.png"
    alt="Input the name of the tag you want to create (e.g. 'New Tag')"
    width="500"
    height="100"
  />
</Frame>

4. Click on the "Create Link" button to assign the tag to the link.

## How to edit or delete a Tag

Tags can be managed via the **Tags settings page** in your AffEasy workspace.

<Frame>
  <img
    className="rounded-lg border border-gray-100"
    src="/images/tags/edit-tags-page.png"
    alt="Editing tags in the Tags settings page"
    width="600"
    height="400"
  />
</Frame>

To manage a tag:

1. Click on the `⋮` button next to the tag to open the menu popover.
2. Choose from the following options:
   - Edit the tag
   - Copy the tag's unique ID (for **programmatic API use**)
   - Delete the tag

<Frame>
  <img
    className="rounded-lg border border-gray-100"
    src="/images/tags/tag-menu-popover.png"
    alt="Edit a tag by clicking on the ⋮ icon"
    width="300"
    height="200"
  />
</Frame>

You can also use the following keyboard shortcuts to manage your tags:

| Keyboard Shortcut | Action                   |
| ----------------- | ------------------------ |
| `E`               | Edit a tag               |
| `I`               | Copy the tag's unique ID |
| `X`               | Delete a tag             |

To edit a tag:

1. Click on the "Edit" button in the tag menu popover.
2. In the tag editing modal, you can:
   - Rename the tag
   - Change the color of the tag

## Can I assign multiple tags to a link?

Yes, you can. In the tags section for a given link, you can select multiple tags to assign to the link.

## How many tags can I create?

The number of tags you can create depends on your plan:

- **Free plan**: Up to 5 tags
- **Pro plan**: Up to 25 tags
- **Business plan**: Up to 150 tags
- **Enterprise plan**: Unlimited tags

Check out our [pricing page](https://affeasy.link/pricing) for more information on our plans.
